## 🔧 Features

- 🔐 User authentication (Login / Register)
- 🚘 Browse cars by category
- 📄 Detailed car pages
- ❤️ Add/remove cars from cart
- 🛒 Place orders
- 📷 Image upload via Cloudinary
- 📱 Fully responsive UI
- ⚙️ RESTful API built with Node.js and MongoDB

---

## 🛠️ Tech Stack

### Frontend:
- React.js
- React Router DOM
- Tailwind CSS
- Axios
- React Toastify

### Backend:
- Node.js
- Express.js
- MongoDB + Mongoose
- Multer
- Cloudinary
